Satellite Tracker

This application will allow users to create and launch satellites into an orbit. They can also track satellites (their own but also satellites created by other users) and see if they are visible at their location at a given time. Users can also follow other users and see when these users create or launch satellites.

Entities

The entities used in this application are shown here. The type relates to the type as seen in the application, so it differs a bit from how the entities will be stored. E.g. location is a seperate model in the application, but in the database it will be saved as GeoJSON.

Satellite

  • id: string
  • name: string
  • mass: number
  • radiusOfBase: number
  • colourOfBase: string
  • radiusOfParts: number
  • purpose: string
  • satelliteParts: satellitePart[]
  • orbit: orbit
  • launch: launch
  • createdAt: date
  • lastUpdated: date

Satellite Part

  • id: string
  • name: string
  • function: string
  • material: string
  • colour: string
  • amount: number
  • createdAt: date
  • lastUpdated: date

Orbit

  • id: string
  • type: string
  • inclination: number
  • semiMajorAxis: number
  • period: number
  • eccentricity: number
  • createdAt: date
  • lastUpdated: date

Launch

  • id: string
  • launchDate: date
  • launchSite: location
  • succeeded: boolean
  • createdAt: date
  • lastUpdated: date

User

  • id: string
  • userName: string
  • email: string
  • password: string
  • location: location
  • createdAt: date
  • lastUpdated: date

Location

  • type: string
  • coordinates: number[]
