Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt.

user

  • id: string
  • userName: string
  • email: string
  • password: string
  • location: location
  • createDate: date
  • updateDate: date

location

  • type: string
  • coordinates: number[]

satellite

  • id: string
  • name: string
  • mass: number
  • radiusOfBase: number
  • colourOfBase: string
  • radiusOfParts: number
  • purpose: string
  • satelliteParts: satellitePart[]
  • orbit: orbit
  • launch: launch
  • createDate: date
  • updateDate: date

satellitePart

  • id: string
  • name: string
  • function: string
  • material: string
  • colour: string
  • amount: number
  • createDate: date
  • updateDate: date

orbit

  • id: string
  • type: string
  • inclination: number
  • semiMajorAxis: number
  • period: number
  • eccentricity: number
  • createDate: date
  • updateDate: date

launch

  • id: string
  • dateOfLaunch: date
  • location: location
  • succeeded: boolean
  • createDate: date
  • updateDate: date
