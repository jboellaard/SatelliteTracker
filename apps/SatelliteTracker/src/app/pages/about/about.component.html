Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt.

satellite

  • radiusOfParts: number
  • purpose: string
  • satelliteParts: satellitePart[]
  • orbit: orbit
  • launch: launch
  • createdAt: date
  • updatedAt: date

  • id: string
  • name: string
  • function: string
  • material: string
  • colour: string
  • amount: number
  • createdAt: date
  • updatedAt: date

orbit

  • id: string
  • type: string
  • inclination: number
  • semiMajorAxis: number
  • period: number
  • eccentricity: number

  • updatedAt: date

launch

  • id: string
  • launchDate: date
  • launchSite: location
  • succeeded: boolean
  • createdAt: date
  • updatedAt: date

user

  • id: string
  • userName: string
  • email: string
  • password: string
  • location: location
  • createdAt: date
  • updatedAt: date

location

  • coordinates: number[]
